# Quinn Visual Style Guide

Warm, approachable, professional, modern.

## Brand Personality
- **Warm & approachable**: Kind expression, calm confidence
- **Trustworthy & professional**: Clean lines, minimal noise
- **Curious & enthusiastic**: Subtle glow and lively posture

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Mist/Paper | `#FFFAE9` | Backgrounds |
| Rose | `#C77D7A` | Accent, apparel |
| Slate | `#172133` | Dark mode backgrounds |
