
# Quinn Brand Kit

A lightweight, ready-to-drop brand kit for **Quinn**, a friendly AI assistant identity used across docs, slides, demos, and GitHub Pages.

## What's Included

- **Avatars**: Canonical, Teams-ready sizes, documentation, transparent, and dark-mode variants
- **Banners**: 1600×900 headers for slides or pages
- **Slides**: PowerPoint template with example slides (cover, section, content, image+text, dark mode)
- **Style Guide**: Markdown visual style guide
- **GitHub Pages**: Theme CSS + landing page example

## Quick Start

### GitHub Pages / MkDocs
1. Copy `GitHub_Pages/quinn-theme.css` into your site
2. Wire it in via `mkdocs.yml` (see snippet below)
3. Add the provided landing page or adapt it for your site

### PowerPoint
Open the template in `Slides/` and duplicate slides as needed.
