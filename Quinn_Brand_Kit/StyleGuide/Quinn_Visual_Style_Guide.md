# Quinn — Visual Style Guide (v1)

Quinn is a friendly AI assistant avatar with a modern, minimalist, professional look—warm, curious, and supportive.

## 1) Brand personality
- **Warm & approachable**: kind expression, calm confidence
- **Trustworthy & professional**: clean lines, minimal noise
- **Curious & enthusiastic**: subtle glow and lively posture

## 2) Color palette
**Primary neutrals (backgrounds):**
- Mist / Paper: **#FFFAE9**

**Accent (apparel):**
- Rose (shirt): **#C77D7A** (muted rose)

**Optional supporting tones:**
- Slate (dark mode background): **#172133**

> Guidance: keep colors **muted** and **natural**. Avoid high-saturation oranges/teals or neon gradients.

## 3) Illustration rules
- Clean vector / soft-illustration hybrid
- Subtle glow only (suggest “intelligence” without sci-fi)
- Consistent line weight and soft shading
- Plain background, no props

## 4) Approved asset variants
- **Canonical (source of truth):** `quinn_avatar_canonical_1024.png`
- **Teams/profile:** `quinn_avatar_teams_256.png`, `quinn_avatar_teams_400.png`, `quinn_avatar_teams_512.png`
- **Docs/wiki:** `quinn_avatar_docs_1200.png`
- **Slides (transparent):** `quinn_avatar_slides_transparent_1600.png` (also `quinn_avatar_transparent_1024.png`)
- **Dark mode:** `quinn_avatar_darkmode_1024.png`
- **Banner:** `quinn_avatar_banner_1600x900.png`

## 5) Usage tips
- Prefer the Teams sizes for any UI that renders at small dimensions.
- Use the transparent slide asset when placing on colored shapes or photos.
- For dark backgrounds, use the dark mode variant to preserve readability.

## 6) Naming + storage
Recommended folder structure:

```
Quinn/
  Avatars/
  Banners/
  Slides/
  Docs/
  StyleGuide/
```

## 7) Change control
Keep Quinn recognizable: avoid changing pose/expression; vary only crop, background, and minor color tweaks.
